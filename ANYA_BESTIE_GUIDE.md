## 🚨 Things That Can Go Wrong (And How to Fix Them)

| Problem | Fix |
|---------|-----|
| Gemini returns garbage JSON | Use `responseMimeType: 'application/json'` in config |
| Supabase blocks all queries | Check RLS policies, might need `USING (true)` for public SELECT |
| Images not uploading | Check bucket exists, check storage policies, check file < 5MB |
| WhatsApp URL doesn't open | Use `https://wa.me/` (not `whatsapp://`), include country code `91` |
| Framer Motion crashes | Make sure component has `'use client';` at the top |
| SCSS not applying | Check import path, use `styles.className` not string |
| Auth redirect loop | Check `middleware.js` matcher + Supabase OAuth callback URL |
